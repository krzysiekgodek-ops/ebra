# EBRA Kalkulatory ⚙️

Platforma z zestawem kalkulatorów rzemieślniczych dla pasjonatów domowej produkcji.

## 🚀 O Projekcie
Strona-hub łącząca specjalistyczne kalkulatory dla różnych branż rzemieślniczych, m.in.:
- **Masarz Master** - receptury masarskie.
- **Piekarski Mistrz** - domowe wypieki.
- **Ciastkarski Wirtuoz** - desery i torty.
- **Mistrz Nalewek** - kalkulatory alkoholi.
- **Kalkulator Ramiarza** - oprawa obrazów.
- **Kalkulator Szklarza** - usługi szklarskie.

## 🛠 Technologie
- HTML5 / CSS3 (Artisan Dark Theme)
- JavaScript (Vanilla JS)
- GSAP / Intersection Observer for Animations
